# Problem 1: Array Sum

## Description
Given an array of integers, compute the sum of all elements.

## Input Format
- First line: integer N (number of elements)
- Second line: N space-separated integers

## Output Format
- Single line: "Sum: X" where X is the sum

## Example
**Input:**
5
1 2 3 4 5

**Output:**
Sum: 15

## How to Test
1. Open this folder in Codespaces
2. Compile: `g++ solution.cpp -o solution`
3. Test: `./solution < test_cases/input_01.txt`
4. Compare output to `test_cases/output_01.txt`
